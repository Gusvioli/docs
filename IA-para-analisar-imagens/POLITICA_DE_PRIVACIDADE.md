Política de Privacidade da Extensão Chrome Gemini

Esta Política de Privacidade descreve como a Extensão Chrome Gemini ("nós", "nosso" ou "a Extensão") coleta, usa e compartilha informações quando você usa nossa Extensão para Google Chrome.

1. Informações que Coletamos

A Extensão Chrome Gemini é projetada para ser minimamente invasiva em relação à sua privacidade. Nós não coletamos, armazenamos ou transmitimos quaisquer informações de identificação pessoal (PII) do usuário.

As únicas informações que a Extensão pode processar são:

- **Imagens para Análise:** Quando você usa a extensão para analisar uma imagem (seja colando, capturando uma área da tela ou usando o menu de contexto), a imagem é processada e enviada para a API do Google Gemini para análise. A imagem não é armazenada em nossos servidores.

- **Histórico de Análise:** Para sua conveniência, a extensão salva um histórico de suas análises recentes. Este histórico inclui a imagem analisada, o prompt utilizado e o resultado da análise. **Esses dados são armazenados exclusivamente no seu dispositivo local**, usando a API de armazenamento do navegador (`chrome.storage.local`), e não são transmitidos para nenhum outro lugar. Você pode limpar seu histórico a qualquer momento através da interface da extensão. Além disso, você pode desativar o salvamento do histórico na página de configurações.

- **Configurações da Extensão:** As preferências e configurações que você personaliza na página de configurações da Extensão (como opções de exibição ou comportamento) são armazenadas localmente em seu navegador usando o `chrome.storage.local`. Essas configurações são armazenadas apenas em seu dispositivo e não são acessadas ou transmitidas por nós.

2. Como Usamos as Informações

As informações processadas pela Extensão são utilizadas estritamente para os seguintes propósitos:

- **Para Fornecer a Funcionalidade Principal:** Enviar a imagem fornecida por você para a API do Google Gemini e exibir o resultado.
- **Para Manter o Histórico:** Armazenar suas análises passadas localmente para referência futura.
- **Para Salvar suas Preferências:** Manter suas configurações, como a chave de API e outras opções, salvas entre as sessões.

3. Compartilhamento de Informações

- **Com a API do Gemini:** As imagens que você escolhe analisar são enviadas para a API do Google Gemini. O uso desta API está sujeito à Política de Privacidade do Google (disponível em https://policies.google.com/privacy).
- **Com Terceiros:** Não compartilhamos suas informações com quaisquer outros terceiros. Todos os dados, exceto as imagens enviadas para a API do Gemini durante o uso, permanecem no seu dispositivo.

4. Segurança dos Dados

Tomamos medidas razoáveis para proteger as informações processadas pela Extensão. No entanto, lembre-se de que nenhum método de transmissão pela internet ou de armazenamento eletrônico é 100% seguro.

5. Suas Escolhas e Direitos

Como não coletamos PII em nossos servidores, não há informações pessoais suas para gerenciar em nossos sistemas. Você pode gerenciar as configurações da Extensão (incluindo desativar o histórico) e limpar o histórico armazenado diretamente na interface da extensão. Você pode remover a Extensão a qualquer momento através da página `chrome://extensions/` do Chrome, o que removerá todos os dados armazenados localmente.

6. Alterações a Esta Política de Privacidade

Podemos atualizar nossa Política de Privacidade periodicamente. Notificaremos você sobre quaisquer alterações publicando a nova Política de Privacidade nesta página. É aconselhável revisar esta Política de Privacidade periodicamente para quaisquer alterações. As alterações a esta Política de Privacidade são efetivas quando são publicadas nesta página.
