
# Política de Privacidade para a extensão SPID Delivery

**Última atualização:** 6 de fevereiro de 2026

A sua privacidade é importante para nós. Esta Política de Privacidade explica como a extensão "SPID Delivery" (a "extensão") coleta, usa e protege suas informações.

## Informações que Coletamos

A extensão SPID Delivery foi projetada para funcionar inteiramente no seu navegador. Nenhuma informação pessoal ou de navegação é enviada para servidores externos.

As seguintes informações são armazenadas localmente no seu navegador usando a API `chrome.storage.local`:

*   **Estado dos Pedidos:** A extensão armazena o status e os IDs dos pedidos do site `https://janis.in/` para monitorar alterações, como novos pedidos ou atualizações de status. Esses dados são comparados com os dados da próxima verificação para identificar novidades.
*   **Configurações do Usuário:** Suas preferências, como a ativação/desativação do monitoramento, recarregamento automático da página e notificações por voz (TTS), são salvas para personalizar sua experiência.
*   **Últimos Pedidos Monitorados:** Uma lista dos últimos pedidos que estão sendo monitorados é mantida para fins de exibição e relatório no popup da extensão.

## Como Usamos Suas Informações

As informações coletadas são usadas exclusivamente para as seguintes finalidades:

*   **Monitoramento de Pedidos:** Para verificar a existência de novos pedidos ou mudanças de status no site `https://janis.in/`.
*   **Notificações:** Para lhe enviar notificações no desktop e alertas sonoros (TTS) sobre novos pedidos e atualizações de status.
*   **Exibição de Status:** Para mostrar a contagem de pedidos ativos no ícone da extensão e exibir detalhes no popup.
*   **Personalização:** Para garantir que a extensão funcione de acordo com as configurações que você definiu.

## Compartilhamento de Dados

A extensão SPID Delivery **não compartilha** nenhuma de suas informações com terceiros. Todos os dados coletados permanecem armazenados localmente no seu dispositivo. A extensão não possui um servidor central para onde os dados são enviados. A única comunicação de rede que a extensão realiza é com a URL `https://janis.in/*` para obter os dados dos pedidos, conforme sua função principal.

## Segurança

Tomamos medidas razoáveis para proteger as informações que armazenamos. Como os dados são armazenados localmente em seu navegador, a segurança dos seus dados depende também da segurança do seu próprio computador e do seu navegador.

Podemos atualizar nossa Política de Privacidade de tempos em tempos. Notificaremos você sobre quaisquer alterações, publicando a nova Política de Privacidade nesta página. Recomenda-se que você revise esta Política de Privacidade periodicamente para quaisquer alterações.
