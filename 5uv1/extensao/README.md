## 📄 Políticas e Termos

- **[Política de Privacidade](privacy-policy.md)**  
  Detalha como seus dados são processados localmente, como funciona a criptografia ponta a ponta e garante que não vendemos seus dados.

## 🚀 Sobre o 5uv1

O **5uv1** é um cliente de chat focado em segurança e privacidade, operando diretamente no painel lateral do navegador.

**Principais recursos:**

- 🔒 Criptografia de Ponta a Ponta (E2EE).
- ⚡ Conexão P2P direta via WebRTC.
- 👻 Sem armazenamento de mensagens em servidores centrais.
- 📂 Código aberto e auditável.
