# Política de Privacidade - 5uv1

Última atualização: 14 de março de 2026

A extensão 5uv1 foi criada para fornecer comunicação P2P com foco em privacidade e segurança. Esta política explica quais dados podem ser processados e como eles são utilizados.

## 1. Dados processados pela extensão
A extensão pode processar localmente os seguintes dados:
- Identificador da sessão ou ID do usuário
- Preferências da interface
- Apelidos locais definidos pelo usuário
- Histórico local de mensagens e metadados associados
- Configurações de servidor
- Fingerprints de chaves para verificação de identidade entre pares

## 2. Onde os dados ficam armazenados
Os dados de configuração e histórico local são armazenados no armazenamento local do navegador, no dispositivo do próprio usuário.

## 3. Comunicação com servidores
A extensão pode se conectar a servidores WebSocket locais ou remotos configurados pelo usuário ou distribuídos com a extensão. Esses servidores são usados para sinalização e transporte de mensagens.

Quando a comunicação ponta a ponta está ativa, o conteúdo das mensagens é protegido entre as partes participantes.

## 4. Notificações
A extensão pode exibir notificações locais do navegador para avisar o usuário sobre novas mensagens.

## 5. Compartilhamento de dados
A extensão não vende dados pessoais e não compartilha dados com fins de publicidade.

## 6. Relato de problemas
Caso o usuário deseje reportar bugs ou sugerir melhorias, pode usar o repositório oficial:
https://github.com/Gusvioli/docs/issues

O usuário não deve incluir senhas, chaves privadas ou conteúdo sensível de conversas em relatórios públicos.

## 7. Segurança
A extensão utiliza mecanismos de proteção como criptografia ponta a ponta, validação de origem, sanitização de conteúdo renderizado e verificação de identidade por fingerprint quando aplicável.
